
**CSS Grid**是一种创建网格布局的方式，他的出现作为对flexbox不足的一种补充(一维布局)，可以同时处理列和行。与flexbox类似，通过对 **父容器(Grid Container)和子元素(Grid Items)** 设置相应的CSS Grid规则，可以比较容易的实现网格布局和定义组件内部元素间大小、位置和图层之间的关系。
